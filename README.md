# 使用方式

1.JQuery CDN
```
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
```

2.前往元素
```
<script src="https://github.com/fifisshh/Web_JQuery/blob/main/main.js"></script>
```

## 屬性介紹
屬性名稱          |屬性說明
------------------|-------------
data-gt-target    |前往的元素，需為ID
data-gt-dur       |時間
data-gt-offset    |位移
